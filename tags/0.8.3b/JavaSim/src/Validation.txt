Validation issues for the simulation XML description file

- Cannot be two agent types with the same name (the value of name attribute in AgentType element).
- For a specific agent type we cannot have two self belief properties with the same name (the value of name attribute in SelfBeliefAttribute element)
- For a specific agent type we cannot have two attributes (objective properties) with the same name (the value of the name attribute in Attribute  element)
