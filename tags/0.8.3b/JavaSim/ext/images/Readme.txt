Nimbus is a theme developed by Sun. Since Java 6 Beta 10, Nimbus is one
of the offical platform independent Look&Feel for Java applications.

http://dlc.sun.com/osol/jds/downloads/extras/nimbus-0.0.16.tar.bz2

NOT rename the file names, because then they are not exchangeable anymore.
