Directions for installing AOR-JavaSim

The Java Simulation Management System for Agent-Object-Relationship (AOR) Simulation

Requires the installation of a Java 6 Development Kit from the Java SE download site (choose, e.g., JDK 6 Update 14 or later).

  1. Extract the ZIP file (with the option "use path names") to a folder of your choice (for instance to D:\).
  2. Some of the application modules (e.g. visualization) are OS dependent, therefore you may need to download additional files for such modules. By default, the distribution comes prepared for Windows 32bits version. If you are using another OS, please download the jar module file for the OS where you need to run the AOR-JavaSim, and copy it to the modules directory located on AOR-JavaSim application main folder. You may have multiple versions of the same module in that directory, the application will load only the appropriate version for you, based on the current running OS on your PC.
  3. Start the program AOR-Simulator.exe
  4. You can use a simple text editor for writing AOR simulation scenarios. But using an XML editor has several advantages: 1) you get syntax guidance by syntax highlighting and by auto-completion, 2) you can immediately check your file for syntactic correctness by using the validation function.
  5. A good open source XML editor is the XML Copy Editor. Under Tools/Options you have to uncheck the "Save UTF-8 Byte Order Mark" (this option is on the right bottom part).
  6. Then you can choose this editor as your external editor under View/Preferences in AOR-JavaSim.

After starting the simulator you can open a sample scenario file in the projects directory. After choosing one of the sample scenarios, go to the Build drop down menu and click Build all (alternatively, press Ctrl-B). If there are no error messages, you can then run the sample scenario by clicking the Run button (or pressing Ctrl-R), going either to the Visualization tab or to the Statistics tab for observing the simulation.
