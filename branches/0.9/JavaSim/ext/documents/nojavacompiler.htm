No Java 6 compiler was found.

Make sure you are running this application at least with the Java Standard Developer Kit (SDK) 6. The Java Runtime Environment (JRE) is not sufficient because it does not include a Java compiler.

Please download the Java SDK or JDK from http://java.sun.com and install it on your machine.

Notice without the AOR-Simulator.exe, or AOR-Simulator.sh file, you are only be able to start the AOR-Simulator application by starting the AOR-Simulator.jar file with the java application of the SDK. On a Windows system you need to type the following in the console. 

"c:\Program Files\Java\jdk1.6.0_10\bin\java.exe" -jar AOR-Simulator.jar

The file AOR-Simulator.jar need to be in the same path. Notice, your Java directory is maybe a different one. For more informations, please ask the developer of the application. You'll be able to find contact informations the project website http://aor-simulation.org .
